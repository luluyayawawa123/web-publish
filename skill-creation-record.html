web-publish Skill 制作全记录

用 Skill 展示 Skill 的诞生过程

一、需求来源

用户有一个 TXT 文档，希望做成网页发布到公网，让别人直接点链接就能看。

二、技术方案选型

尝试了多个方案，最终选择 GitHub Pages（稳定）和 Cloudflare Tunnel（临时验证）：

方案结果
surge.sh未安装，需要账号
localtunnel503 不稳定
Cloudflare Tunnel免费、无需账号、但偶尔闪断
GitHub Pages✅ 免费、永久稳定、全球 CDN

三、实施步骤

第一步：找到源文件

find ~/Documents -name "小米"

第二步：生成 HTML 页面

编写 generate_page.py 脚本，支持 5 种风格自动选择：

  • tech：技术文档，暗色主题
  • story：散文叙事，衬线字体
  • product：产品营销，渐变背景
  • data：数据报告，表格突出
  • clean：通用简洁

第三步：发布到 GitHub Pages

git add → git commit → git push → 自动构建 → 链接生效

四、Skill 设计思路

把流程固化下来，用户只需要说"做成网页"，AI 自动执行全部步骤。

Skill 结构

~/.openclaw/skills/web-publish/
├── SKILL.md              # 触发条件 + 流程说明
└── scripts/
    └── generate_page.py  # HTML 生成脚本

五、踩过的坑

  1. localtunnel 返回 503 — 换 Cloudflare Tunnel
  2. cloudflared 下载 URL 错误 — 官方发布的是 .tgz 压缩包
  3. QUIC 连接静默断开 — cloudflared 进程不退出但已不通
  4. 最终方案 — GitHub Pages，不存在进程问题

六、最终效果

用户获得一个永久链接：

https://luluyayawawa123.github.io/web-publish/

  • 免费、无需注册
  • 手机电脑都能看
  • 链接永远不变
  • 全球 CDN，国内可访问
