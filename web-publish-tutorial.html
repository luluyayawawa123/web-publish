OpenClaw 网页发布 Skill 手把手教程

免费 · 无需账号

一篇手把手教程，让你的 AI 助手能把任何内容做成网页，免费发布到公网。

一、效果预览

对你的 AI 说："把这个文档做成网页发给我"——它会自动生成一个美观的 HTML 页面，通过 GitHub Pages 发布到公网，给你一个可以直接分享的链接。

全程免费，无需注册任何账号（只需要一个 GitHub 账号）。

二、Skill 目录结构

~/.openclaw/skills/web-publish/
├── SKILL.md              # Skill 定义文件（触发条件 + 流程说明）
└── scripts/
    └── generate_page.py  # HTML 生成脚本

三、SKILL.md 核心内容

---
name: web-publish
description: >
  将内容发布为公网可访问的网页。
  支持从 TXT / Markdown / HTML 生成美观的 HTML 页面；
  通过 GitHub Pages 免费发布到公网；
  返回一个可以直接分享的链接。
  触发词：做成网页、发布到公网、生成网页链接
---

四、generate_page.py 核心功能

  • 自动识别 Markdown vs 纯文本
  • 5 种预设风格：tech / story / product / data / clean
  • 根据内容自动选择最佳风格
  • 暗色/亮色主题，移动端适配
  • 一键生成完整 HTML 文件

五、GitHub Pages 发布流程

  1. 生成 HTML 页面
  2. 复制到仓库目录 ~/Documents/web-publish-repo/
  3. git add . && git commit -m "publish" && git push
  4. 1-2 分钟后链接生效

链接格式固定：https://<用户名>.github.io/web-publish/index.html

六、Cloudflare Tunnel 原理（备选方案）

本机主动向 Cloudflare 建立出站连接（QUIC/UDP），公网用户访问 Cloudflare 分配的域名，请求通过隧道转发到本机。

优点：无需 GitHub 账号，一行命令搞定。

缺点：quick tunnel 不够稳定，偶尔闪断。

七、常见问题

Q: 链接能保持多久？

A: GitHub Pages 永久有效。Cloudflare Tunnel 取决于进程存活时间。

Q: 有流量限制吗？

A: GitHub Pages 免费版 1GB 带宽/月，日常分享完全够用。

Q: 国内能访问吗？

A: GitHub Pages 国内可直接访问。Cloudflare 的 .trycloudflare.com 也可以。
