一、需求来源

用户有一个 TXT 文档（小米 Token Plan 接入龙虾要点），希望：

  • 做成网页
  • 发布到公网
  • 免费
  • 别人直接点链接就能看

二、技术方案选型

尝试了以下方案，最终成功：

方案结果
surge.sh未安装，需要账号
Vercel CLI未安装
Netlify CLI未安装
localtunnel (npx)能获取链接但 503 不稳定
\1✅ 免费、无需账号、稳定

三、实施步骤

第一步：找到源文件

find ~/Documents -name "\1Talk\1" -o -name "\1"

找到：~/Documents/New project/小米Token Plan接入龙虾要点.txt

第二步：读取内容

直接 read 文件，获取 TXT 全文内容。

第三步：生成 HTML 页面

编写了一个 Python 脚本 generate_page.py，核心功能：

  • 自动识别 Markdown vs 纯文本
  • 暗色主题，移动端适配
  • 代码块、列表、表格、引用等格式支持
  • 一键生成完整 HTML

第四步：启动本地 HTTP 服务

python3 -m http.server 3456 -d <output_dir>

第五步：下载 cloudflared（一次性）

curl 下载 cloudflared 二进制到 /tmp/cloudflared

无需安装、无需配置、无需账号

第六步：启动 Cloudflare Tunnel

/tmp/cloudflared tunnel --url http://localhost:3456

输出中提取 *.trycloudflare.com 链接

第七步：返回链接

用户直接获得公网可访问的 URL

四、Skill 设计思路

为什么做成 Skill？

把这套流程固化下来，以后用户只需要说"做成网页"，AI 就自动执行全部步骤，不需要重新思考技术方案。

Skill 结构

web-publish/

├── SKILL.md # 触发条件 + 流程说明

└── scripts/

└── generate_page.py # HTML 生成脚本

SKILL.md 包含什么

  • \1: name + description（决定何时触发）
  • \1: 7 步标准流程
  • \1: TXT / Markdown / HTML / 飞书文档 各怎么处理
  • \1: cloudflared 位置、端口策略、链接时效

generate_page.py 做什么

  • 接收 --title、--input、--output、--theme 参数
  • 自动判断内容是 Markdown 还是纯文本
  • Markdown → 转 HTML（标题/列表/代码块/表格/引用）
  • 纯文本 → <pre> 原样保留
  • 输出带暗色主题样式的完整 HTML 文件

五、踩过的坑

  1. \1 — 换 Cloudflare Tunnel 解决
  2. \1 — 官方发布的是 .tgz 压缩包，不是裸二进制
  3. \1 — 改存到 /tmp 目录
  4. \1 — 换端口或先检查

六、最终效果

用户获得一个链接，类似：

https://province-question-intelligence-fairy.trycloudflare.com

  • 免费、无需注册
  • 手机电脑都能看
  • 暗色主题、排版美观
  • 只要本机服务运行着，链接就有效

七、扩展用法

这个 Skill 不只用于 TXT，还可以：

  • 把飞书文档做成网页分享
  • 把代码片段做成可分享的网页
  • 把 Markdown 笔记快速发布
  • 临时给同事展示方案/配置/文档
